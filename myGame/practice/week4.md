- 원형 / AABB / OBB Bounding Box 시각화
- 세 방식의 충돌 판정 차이 관찰

관찰 내용
원형 vs AABB
원형은 중심 거리로 판단해서 모서리에서도 둥글게 충돌이 생겨 실제보다 더 빠르거나 늦게 충돌한 것처럼 보일 수 있다
AABB는 사각형 모서리를 기준으로 딱딱하게 충돌해서 모서리 판정이 더 정확하지만 각이 있어 충돌이 되지 않게 보여도 충돌 판정이 적용될 수 있다

AABB vs OBB
AABB는 회전해도 축이 고정이라 회전된 물체를 표현하지 못한다
OBB는 물체의 회전을 반영해서 회전하는 이미지와 거의 동일하게 충돌 판정이 적용된다

AI와의 대화에서 배운 것
OBB: 꼭짓점을 회전시켜 만든 뒤 여러 축으로 투영해서 겹침 여부로 충돌을 판단한다
회전 처리: Rect 기준이 아니라 각도 기반으로 꼭짓점을 계산해야 스프라이트와 OBB가 완전히 같이 회전한다

내 게임에 적용한다면
기본적인 오브젝트는 AABB로 처리하여 연산량을 줄이고 중요한 오브젝트나 정확한 판정이 필요한 오브젝트의 경우에 OBB를사용할 것이다
